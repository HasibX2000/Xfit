Installation

  1. Extract the downloaded zip file
  2. Upload the contents to your web server
  3. Navigate to your domain

Modifications

1. Changing Content

To modify the template content, look for these sections in HTML/index.html:

  • Header/Logo: Update the logo image in header section
  • About Section: Modify text and image in #about section
  • Services: Edit cards in #services section
  • Schedule: Update class timings in #schedule section
  • Testimonials: Modify reviews in #testimonials section
  • Footer: Update contact info and links in footer section

2. Styling Changes

Main styling variables are located in assets/css/style.css:

:root {
    /* Colors */
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --text-color: #334155;
    --body-color: #ffffff;
    
    /* Typography */
    --body-font: 'Inter', sans-serif;
    --heading-font: 'Inter', sans-serif;
    
    /* Spacing */
    --section-spacing: 100px;
    --container-width: 1320px;
}

3. Image Replacement

Replace images in the following directories:

  • assets/img/gallery/ - Gallery images
  • assets/img/programs/ - Program/service images
  • assets/img/testimonials/ - Testimonial profile pictures
  • assets/img/logo.png - Main logo
  • assets/img/favicon.png - Browser favicon

Recommended image dimensions:

  • Logo: 180x60px
  • Gallery: 800x600px
  • Programs: 600x400px
  • Testimonials: 100x100px (square)

4. Contact Information

Update contact details in these locations:

  • Header phone number
  • CTA section contact button
  • Footer contact information
  • Social media links in footer

5. Custom JavaScript

Modify functionality in assets/js/script.js:

  • Mobile menu behavior
  • Smooth scrolling
  • Testimonial slider settings
  • Animation triggers
  • Schedule tab functionality

6. Performance Tips

  • Optimize images before uploading
  • Minify CSS and JavaScript files for production
  • Enable browser caching for assets
  • Use appropriate image formats (WebP when possible)
  • Lazy load images below the fold
